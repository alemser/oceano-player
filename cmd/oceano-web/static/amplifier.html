Inputs

Top-to-bottom is the amplifier's real cycle order. In cycle mode, the next input command moves downward through this list and wraps back to the top, so keep this order aligned with the amplifier's clockwise input sequence. Index 0 is the default input. Internal ID stays hidden; edit only logical name and visibility.

Connected Devices

Register the physical devices connected to each input. Their names replace raw input names in the input selector. A device can span multiple inputs and doesn't need to have a remote control.
