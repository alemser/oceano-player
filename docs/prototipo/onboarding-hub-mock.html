First-run checklist

  1. Run sudo oceano-setup on the Pi (DAC, capture, AirPlay/BT basics)
  2. Choose capture device (REC OUT → USB interface)
  3. Add ACRCloud credentials (recognition)
  4. Optional
  5. Optional: amplifier IR (Broadlink → IR learn) — or skip IR and still map inputs & devices for calibration / UI
  6. Run calibration for physical media inputs only

Streaming (AirPlay name, BT) stays collapsible or a single short row below physical steps.
